# Hatom

Connect Hatom to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/hatom) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-hatom@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-hatom --url https://cortex.joai.ai/mcp/apps/hatom
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/hatom.git ~/.cursor/plugins/local/joai-hatom
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-hatom&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktaGF0b20iOnsidXJsIjoiaHR0cHM6Ly9jb3J0ZXguam9haS5haS9tY3AvYXBwcy9oYXRvbSJ9fX0=) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/hatom
```
